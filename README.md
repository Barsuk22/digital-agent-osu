# 🎮 Digital Agent OSU Project

## Описание

Это проект по созданию обучающегося AI-агента, который учится играть в osu! на реальных картах.

Цель — не просто сделать бота, а обучить агента через reinforcement learning:

* двигаться по карте
* находить объекты
* нажимать в нужный момент
* постепенно формировать устойчивые игровые навыки

---

## Текущее состояние

Проект прошёл **Phase 0 — Core Foundation**.

Реализовано:

* полноценный `.osu` parser
* игровая среда (environment)
* система попаданий (judgement)
* live viewer (pygame)
* replay система
* базовый loop взаимодействия агента с игрой

Агент уже:

* двигается
* находит цели
* кликает
* иногда попадает

---

## Как запустить

### ▶ Live просмотр (агент играет в реальном времени)

```
python -m src.apps.live_viewer_osu
```

---

### ▶ Просмотр реплея

```
python -m src.apps.replay_osu
```
