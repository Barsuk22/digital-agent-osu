# Viewer Specification

## Назначение

Viewer показывает, как агент играет.

---

## Возможности

* карта
* курсор
* хвост курсора
* клики (визуальные эффекты)
* попадания (300/100/50/miss)
* combo / accuracy
* replay

---

## Визуальные элементы

### Cursor

* основной кружок
* glow
* хвост (trail)

### Trail

* плавная линия
* увеличивается к курсору
* полупрозрачная

### Click

* вспышка
* кольца
* расширение

---

## Slider

* тело
* шарик
* ticks
* repeat arrows

---

## Config (viewer.yaml)

Можно менять:

* цвета
* длину хвоста
* размеры окна
* fps

---

## Ограничения

* не 1:1 osu! lazer
* упрощённые эффекты
* без точного input lag
