# Skill Extraction Gate

Этот gate пока не является активной стадией. Он нужен позже, когда поведение агента станет достаточно стабильным, чтобы извлекать повторяемые навыки, а не случайные удачные куски.

## Цель

Проверить, что агент играет не хаотично, а повторяемо.

## Что нужно проверять

- hit rate стабилен между eval-запусками;
- useful clicks не разваливаются;
- timing drift не хаотичный;
- успешные паттерны повторяются;
- хорошие эпизоды похожи друг на друга;
- движение не состоит из случайных резких импульсов.

## Критерий прохождения

- агент показывает повторяемое поведение;
- успешные куски можно кластеризовать;
- извлечённые паттерны выглядят как реальные заготовки навыков;
- skill extraction не подменяет обучение случайной удачей.

## Когда возвращаться к этому документу

После стабилизации Phase 1.5 / Movement Polishing и первых шагов multi-map eval.
